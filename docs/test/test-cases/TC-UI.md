**테스트 절차:**

### TC-UI-011-03: TailwindCSS 스타일 적용

| 항목 | 내용 |
|------|------|
| **테스트 ID** | TC-UI-011-03 |
| **요구사항 ID** | FR-UI-011 |
| **우선순위** | P0 |
| **테스트 유형** | Visual Regression |
| **사전 조건** | - TailwindCSS 설정 완료<br>- Playwright 스크린샷 비교 환경 준비<br>- 기준 스크린샷 존재 |

**테스트 절차:**

1. 주요 화면(메인, 설정, 모델 매니저 등)을 Playwright로 캡처한다.
2. 캡처된 스크린샷을 기준 이미지와 비교한다.
3. TailwindCSS 유틸리티 클래스가 올바르게 적용되었는지 확인한다.
4. 라이트/다크 테마 모두에서 스타일을 확인한다.
5. 반응형 레이아웃이 다양한 윈도우 크기에서 올바른지 확인한다.

**기대 결과:**
- 스크린샷이 기준 이미지와 허용 오차(0.1%) 내에서 일치한다.
- TailwindCSS 유틸리티 클래스가 정상 적용된다.
- 라이트/다크 테마 전환 시 모든 요소의 스타일이 올바르다.
- 반응형 레이아웃이 최소 너비(800px)부터 최대 너비까지 올바르게 동작한다.

| **자동화 여부** | Y |

### TC-MOB-001-01: Tier 1 whisper tiny 온디바이스 전사

| 항목 | 내용 |
|------|------|
| **테스트 ID** | TC-MOB-001-01 |
| **요구사항 ID** | FR-MOB-001 |
| **우선순위** | P1 |
| **테스트 유형** | 기능 |
| **사전 조건** | - whisper tiny 모델이 모바일 디바이스에 설치 완료<br>- 테스트용 오디오 파일(30초, 한국어) 준비<br>- iOS 또는 Android 디바이스 |

- 전사 결과의 정확도가 기준치(WER 30% 이하) 이내이다.
